# Mobile JUKE — мобильная аркадная игра «Жуки»

Android-приложение в жанре аркады: насекомые бегают по экрану, задача игрока — уничтожить как можно больше, набирая очки. За промахи — штраф. Проект демонстрирует работу с ключевыми компонентами Android-разработки: от UI и анимаций до баз данных, сенсоров и сетевых запросов.

## Возможности

- **Аркадный геймплей** — насекомые разных видов перемещаются по экрану с настраиваемой скоростью; тап по жуку уничтожает его и начисляет очки, промах — штраф
- **Регистрация игрока** — форма с ФИО, полом, курсом, уровнем сложности, датой рождения и автоматическим определением знака зодиака
- **Система бонусов** — каждые 15 секунд появляется бонус: при активации насекомые начинают реагировать на наклон телефона (акселерометр) и «скатываются» к краю экрана
- **Золотой таракан** — каждые 20 секунд появляется особый жук, очки за которого начисляются пропорционально текущему курсу золота ЦБ РФ (загружается через Retrofit)
- **База данных (Room)** — сохранение результатов, регистрация пользователей, таблица рекордов
- **Виджет на рабочем столе** — отображает актуальный курс золота ЦБ РФ
- **Настройки игры** — скорость, максимальное количество насекомых, интервал бонусов, длительность раунда
- **Правила игры** — загружаются из HTML-ресурсов приложения
- **Список авторов** — кастомизированный ListView с фотографиями и именами

## Стек технологий

| Технология | Применение |
|---|---|
| **Java** | Основной язык разработки |
| **Android SDK** | UI-компоненты: TabLayout, RecyclerView, SeekBar, CalendarView и др. |
| **Room** | Локальная база данных для хранения пользователей и рекордов |
| **Retrofit** | Сетевые запросы к API ЦБ РФ для получения курса золота |
| **ViewModel + Koin** | Сохранение состояния при повороте экрана, dependency injection |
| **SensorManager** | Работа с акселерометром для управления движением насекомых |
| **Gradle (Kotlin DSL)** | Система сборки |

## Запуск

**Требования:** Android Studio, JDK 17+, Android SDK

```bash
git clone https://github.com/Klipo07/Mobile_JUKE.git
```

Откройте проект в Android Studio: **File → Open → выбрать папку Mobile_JUKE**.

Или соберите из терминала:

```bash
cd Mobile_JUKE
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/`.

## Чему учит этот проект

Проект охватывает ключевые аспекты Android-разработки:

- Работа с множественными Activity и Fragment через TabLayout
- Кастомизация стандартных UI-компонентов (ListView с адаптером)
- Хранение данных: SharedPreferences и Room (SQLite)
- Сетевые запросы через Retrofit + парсинг XML (SXML API ЦБ РФ)
- Работа с аппаратными сенсорами (акселерометр)
- App Widgets для рабочего стола
- Архитектура ViewModel + DI через Koin
- Корректная обработка жизненного цикла (поворот экрана)
