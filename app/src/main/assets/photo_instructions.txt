ИНСТРУКЦИЯ ПО ДОБАВЛЕНИЮ ФОТОГРАФИИ АВТОРА
===============================================

Чтобы добавить свою фотографию в приложение:

1. ПОДГОТОВКА ФОТОГРАФИИ:
   - Размер: рекомендуется 200x200 пикселей или больше
   - Формат: JPG, PNG
   - Качество: хорошее, но не слишком большое (до 1MB)

2. СПОСОБ 1 - ЧЕРЕЗ РЕСУРСЫ ПРИЛОЖЕНИЯ:
   - Поместите файл с именем "kirill_photo.jpg" в папку app/src/main/res/drawable/
   - Переименуйте файл в "author1.jpg" (заменив существующий)
   - Пересоберите приложение

3. СПОСОБ 2 - ЧЕРЕЗ ВНЕШНЕЕ ХРАНИЛИЩЕ:
   - Поместите фото в папку Pictures на устройстве
   - Путь должен быть: /storage/emulated/0/Pictures/kirill_photo.jpg
   - Обновите код в AuthorsFragment.java, добавив путь к фото

4. СПОСОБ 3 - ПРОГРАММНОЕ ДОБАВЛЕНИЕ:
   - Используйте камеру устройства для съемки
   - Сохраните фото в папку приложения
   - Обновите путь в коде

ТЕКУЩИЙ СТАТУС:
- Огарков Кирилл Алексеевич - первый в списке авторов
- Использует специальную иконку с коронкой
- Готов к добавлению реальной фотографии

Для тестирования можно использовать любую фотографию с именем "author1.jpg"
